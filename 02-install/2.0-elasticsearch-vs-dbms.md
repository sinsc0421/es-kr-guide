# 2.0 ElasticSearch vs DBMS



| **Relational DB** | **Elasticsearch** |
| ----------------- | ----------------- |
| 데이터베이스(Database)  | 인덱스(Index)        |
| 테이블(Table)        | 타입(Type)          |
| 행(Row)            | 문서(Document)      |
| 열(Column)         | 필드(Field)         |
| 스키마(Schema)       | 매핑(Mapping)       |



| **Relational DB** |  **Elasticsearch** |
| ----------------- | ------------------ |
| SELECT            | GET                |
| UPDATE            | PUT                |
| INSERT            | POST               |
|  DELETE           | DELETE             |
